# Diwali-Sales-Analysis-using-Python
This project performs an in-depth Exploratory Data Analysis (EDA) on a Diwali sales dataset to uncover customer purchasing behavior and sales trends. Using Python libraries such as NumPy, Pandas, Matplotlib, and Seaborn, the dataset was cleaned, processed, and analyzed to generate meaningful business insights.

## Objective
- Perform data cleaning
- Analyze customer demographics
- Identify top-selling products
- Discover purchasing patterns
- Generate business insights

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning

- Removed unnecessary columns
- Handled missing values
- Converted datatypes
- Checked duplicate records

## Exploratory Data Analysis

The following analyses were performed:

- Gender-wise purchases
- Age Group Analysis
- State-wise Sales
- Occupation Analysis
- Marital Status Analysis
- Product Category Analysis
- Top Selling Products

## Key Insights

- Female customers spend more than male customers.
- Customers aged 26–35 contribute the highest sales.
- Uttar Pradesh, Maharashtra, and Karnataka generate the highest revenue.
- IT, Healthcare, and Aviation professionals spend more.
- Food, Clothing, and Electronics are among the top-selling categories.
